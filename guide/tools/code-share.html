carbon.now.sh

carbon.now.sh 是一个代码分享工具，可以将代码转换为图片，支持多种主题、字体、背景等设置。

gist.github.com

gist.github.com 是 GitHub 提供的代码片段分享服务，可以将代码片段保存为 Gist，支持多种语言、Markdown 渲染等功能。

paste.ofcode.org

paste.ofcode.org

compiler-explorer.com

Code Explorer from godbolt，是一个在线编译器，可以将 C/C++ 代码转换为汇编代码，查看结果，支持多种编译器、优化等设置，还能生成链接分享。
